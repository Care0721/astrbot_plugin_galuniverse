# GalUniverse Pro - Galgame 沉浸式助手

**一个专为 Galgame（视觉小说）玩家打造的沉浸式助手插件**

让你的机器人化身 Galgame 陪伴者，每天为你抽取今日老婆、查询圣地巡礼地点，还能随时管理你的 Gal 角色与巡礼数据库。

### ✨ 核心特性
- **今日老婆**：每天固定抽取一位 Galgame 角色 + 运势签文，增加仪式感和陪伴感
- **圣地巡礼**：快速查询 Galgame 相关的真实取景地/圣地信息（支持自定义）
- **完全网页端管理**：无需修改任何代码或 TXT 文件，所有数据（老婆列表、圣地巡礼）都可在 AstrBot 网页后台可视化添加、删除、修改
- **一键重载**：修改配置后立即生效，无需重启机器人
- 支持自定义大量角色与巡礼地点，打造属于你自己的 Gal 宇宙

### 📸 功能预览
- `/今日老婆` —— 每日一抽，签运 + 角色 + 温馨建议
- `/圣地巡礼 CLANNAD` —— 查询游戏对应的圣地信息
- `/重载Gal数据` —— 配置修改后立即刷新数据

### 📖 使用说明

#### 1. 安装插件
- 将插件文件夹放入 AstrBot 的 `data/plugins/` 目录
- 重启 AstrBot（或通过插件市场安装）

#### 2. 配置插件（推荐方式）
1. 打开 AstrBot 网页管理后台（通常是 `http://你的服务器IP:6185`）
2. 进入 **插件管理** → 找到 **Galgame 沉浸式助手**（或你的插件显示名）
3. 点击 **配置**，你会看到以下可编辑项：

   - **是否启用「今日老婆」运势功能**（开关）
   - **今日老婆角色列表**（数组）：点击添加即可输入角色名（如 `古河渚 (CLANNAD)`），支持无限添加/删除/修改
   - **圣地巡礼地点列表**（数组）：每条可填写：
     - `name`：游戏名（用于指令匹配）
     - `desc`：地点描述
     - `link`：详情链接（可选）

4. 修改完成后点击保存

#### 3. 使用指令

| 指令          | 示例                          | 说明                                      |
|---------------|-------------------------------|-------------------------------------------|
| 今日老婆      | `/今日老婆`                   | 抽取今日 Gal 运势（每日固定，同一个人同一天结果一致） |
| 圣地巡礼      | `/圣地巡礼 CLANNAD`           | 查询游戏的圣地巡礼信息                    |
| 重载Gal数据   | `/重载Gal数据`                | 修改网页配置后执行此命令，使数据立即生效 |

**提示**：
- 圣地巡礼指令支持模糊匹配（如输入 `clannad` 或 `CL` 也能匹配）
- 第一次使用建议添加几个常用角色和圣地后再测试
- 所有数据持久化保存在 `data/config/你的插件名_config.json` 中，可备份此文件

#### 4. 数据示例（初始默认值）
**老婆列表** 示例：
- 古河渚 (CLANNAD)
- 立华奏 (Angel Beats!)
- 五更琉璃 (我的妹妹不可能那么可爱)

**圣地巡礼** 示例：
- name: CLANNAD  
  desc: 京都动画圣地 / 古河家面包店  
  link: https://xxx
